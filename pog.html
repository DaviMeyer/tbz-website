Pong game

I wrote this game during my basic training year with the support of my coaches. As a group, we were given the assignment to program our own game in Python. I decided to program a classic from the seventies, namely the game Pong. When you start the game, a window appears that prompts you to enter the size of your screen, and also asks you which game mode you want to play and how many lives each player should have.Once you have finished with the inputs, the game begins and continues until one player has no lives left. At the end, the winner is announced. Don't expect too much, the game still has a few errors and bugs ;) Please be confident that the file is completely safe to play. In the unlikely event that your device displays a warning message, simply click on "more information" and then select "run anyway." Rest easy knowing that the file is completely virus-free and you won't encounter any issues.
